# CommerceHub Enterprise V5 — Sprint 21 Smart Category Engine

Consulta a categoria e seus atributos no Mercado Livre, salva as exigências no Supabase, sugere valores a partir do Product Master, mostra campos faltantes e injeta os atributos no payload de publicação.

## Instalação
1. Suba os arquivos no GitHub Web.
2. Execute `sprint21_smart_category_engine.sql` em uma nova query no Supabase.
3. Teste `/api/smart-category/status`.
4. Abra `/smart-category-engine/category?category_id=MLB1648`.
5. No anúncio, clique em **Atributos inteligentes**.
