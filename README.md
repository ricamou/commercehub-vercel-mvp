# CommerceHub Enterprise V5 — Sprint 27 Marketplace Rules Engine

## Entregue

- árvore de decisão por categoria e marca;
- contrato de payload;
- atributos obrigatórios, condicionais e opcionais;
- localização item x variação;
- estratégia value_id x value_name;
- deduplicação de bloqueios;
- snapshot persistente no Supabase;
- tela Rules Engine;
- integração com Publishing Lab.

## Instalação

1. Envie todos os arquivos pelo GitHub Web.
2. Aguarde o deploy.
3. Abra `/api/health`.
4. Confirme:
   `enterprise-v5-sprint27-marketplace-rules-engine`
5. Execute `Sprint27_Marketplace_Rules_Engine.sql` em uma nova query.
6. Abra o anúncio.
7. Clique em `Rules Engine`.
8. Corrija as ações obrigatórias.
9. Volte ao Publishing Lab.
10. Publique apenas quando estiver PRONTO.
